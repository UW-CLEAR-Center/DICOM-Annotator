Zooming and Changing Window/Level

  How to zoom

    * Put the mouse cursor on the image; scrolling the mouse wheel will zoom.

    * Right click, hold, and move the mouse to drag the image. 

    * Double right clicking will restore the image to the original size.

  How to change window level

    * Put the mouse cursor on the image; shift + moving mouse up/down/left/right will change window/level.

    * Shift + double right click will restore the original window/level.

    * If you want to move your mouse to do other operations instead of change window level, please release the “shift” key

  Note

    * The "Reset View" button can restore to both the original size and window level.


Some Keyboard Shortcuts

    * [: previous image

    * ]: next image

    * enter: move to the next level

    * backspace (PC) or delete (Mac): deletes the last labelled point

    * a & d, w & s: adjust window/level
