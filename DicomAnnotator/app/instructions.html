General Steps for Labeling An Image

   * Check if the image is readable. If unreadable (poor quality, not L-spine or T-spine lateral), click the “Drop the Image” button and go to the next image.

    * Check the orientation of the image. Patient in the image should face reader’s left. Click the "Flip the Image" button if it is facing right.

    * Figure out what the lowest vertebral body is and activate the associated tab in the table.

    * Check if there is hardware/metal overlapping over the current vertebral body. If yes, click the "Hardware/Metal" checkbox.

    * Put four corner points, in the order SupPost->SupAnt->InfAnt->InfPost, for each vertebral body, from the lowest one to the highest one.

    * Decide on a fracture label for each vertebral body.

    * Decide whether the patient has osteoprosis. Click related ratio button on top right.


Zooming and Changing Window/Level

  How to zoom

    * Put the mouse cursor on the image; scrolling the mouse wheel will zoom.

    * Right click, hold, and move the mouse to drag the image. 

    * Double right clicking will restore the image to the original size.

  How to change window level

    * Put the mouse cursor on the image; shift + moving mouse up/down/left/right will change window/level.

    * Shift + double right click will restore the original window/level.

    * If you want to move your mouse to do other operations instead of change window level, please release the “shift” key

  Note

    * The "Reset View" button can restore to both the original size and window level.


Some Keyboard Shortcuts

    * [: previous image

    * ]: next image

    * enter: move to the next level

    * backspace (PC) or delete (Mac): deletes the last labelled point
